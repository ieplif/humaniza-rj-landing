# Humaniza RJ — Landing Page

Site institucional da **Humaniza RJ**, hospedado em [humanizarj.com.br](https://humanizarj.com.br).

## Stack

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Tailwind CSS](https://tailwindcss.com/)

## Como rodar localmente

```bash
# 1. Clone o repositório
git clone https://github.com/ieplif/humaniza-rj-landing.git

# 2. Entre na pasta do projeto
cd humaniza-rj-landing

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

## Build para produção

```bash
npm run build
```

Os arquivos gerados ficam na pasta `dist/`.

## Deploy

O site é hospedado via **GitHub Pages** com domínio personalizado em [humanizarj.com.br](https://humanizarj.com.br).
